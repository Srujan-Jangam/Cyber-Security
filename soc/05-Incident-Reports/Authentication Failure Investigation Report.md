## Incident Title

Authentication Failure Detected on Local System

## Summary

Multiple failed login attempts were observed on the system after intentionally entering incorrect passwords. These events were recorded in system authentication logs, confirming that failed login activity is properly captured and available for investigation.

---

## Log Sources

- `journalctl` (system-wide log aggregation) 
- `lightdm` logs (GUI authentication manager)

---

## Event Description

The failed authentication events occurred when incorrect credentials were entered during the login process. Log entries indicated authentication failure messages generated by the login service.
